SciViews Box

The SciViews Box, or svbox, is an easily installable software environment for reproducible data science with R, Python and Julia.

SciViews::R

An extension to the Tidyverse SciViews::R is an extension to the Tidyverse aiming to make R even more easy to use by everyone.

Zoo/PhytoImage

Zoo/PhytoImage analyzes plankton samples through image analysis and machine learning. Plankton images from various origins can be used. Analyze your plankton through digitized images Plankton samples are traditionally collected with plankton nets and analyzed by taxonomists using a stereomicroscope and determination keys.
